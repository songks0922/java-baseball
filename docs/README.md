# 🔍 기능 목록

## 입력 처리
- 입력을 받는 함수
- 서로 다른 숫자를 입력했는지 판별해주는 함수
- 3자리수가 아닌 다른 자리수가 들어왔는지 확인하는 함수
## 게임 흐름 처리
- 게임을 시작하는 함수
- 숫자를 다 맞췄을 때 게임을 종료시키는 함수
- 게임 종료 시 프로그램을 완전 종료 시키는 함수
- 게임 종료시 게임을 다시 시작 시키는 함수
## 출력 처리
- 컴퓨터의 난수를 생성해주는 함수
- 스트라이크, 볼, 낫싱을 판별해주는 함수
- 결과를 출력해주는 함수
## 예외 처리
- 예외 발생시 애플리케이션을 종료시키는 함수
